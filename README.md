# FlashDocusaurus

A VS Code extension for Docusaurus documentation development with intelligent features.

## Features

- Automatic Docusaurus project detection
- Basic command support

## Requirements

- VS Code 1.73.0 or higher
- A Docusaurus project (with `docusaurus.config.js` or `docusaurus.config.ts`)

## Getting Started

1. Open a Docusaurus project in VS Code
2. The extension will automatically detect your project
3. Use Command Palette (Ctrl+Shift+P / Cmd+Shift+P) and search for "FlashDocusaurus: Hello World"

## Development

```bash
# Install dependencies
npm install

# Compile the extension
npm run compile

# Watch for changes
npm run watch
```
